# 小程序反编译工具 (AppletDeAnalyze)

一个基于 Node.js + pywebview 的微信小程序反编译工具。

## 功能特性

- **可视化界面**：使用 pywebview 提供友好的图形界面
- **实时目录监控**：自动检测目录变化，实时更新小程序列表
- **多参数配置**：支持自定义输出目录、代码格式化等参数


## 系统要求

- **Python**：3.8+
- **Node.js**：12+


## 安装方法

### 方法 1：使用打包版本

1. 下载打包后的可执行文件
2. 直接运行 `AppletDeAnalyze_v1.2.exe`，该方法不支持缓存，每次启动都需要指定小程序目录。

### 方法 2：从源码运行（推荐）

1. 克隆或下载项目到本地
2. 安装依赖：
   ```bash
   # 安装 Python 依赖
   pip install pywebview
   
   # 安装 Node.js 依赖
   npm install
3. 运行应用

   ```python app.py```

## 使用步骤

### 1.选择小程序目录
小程序目录通常位于：C:\Users\用户名\AppData\Roaming\Tencent\xwechat\radium\Applet\packages

### 2.反编译小程序
在小程序列表中选择目标小程序，点击「反编译」按钮。等待反编译完成，成功后会显示「✓ 已反编译」标记，反编译结果默认保存在 output 目录中。

### 3.查看反编译结果
点击「打开」按钮，会自动打开对应的反编译输出目录，查看反编译后的代码和资源文件。

### 4. 管理小程序
- 删除Appid目录 ：删除源小程序目录
- 删除反编译目录 ：删除反编译输出目录（同时移除「已反编译」标记）
- 
### 5. Js分析
勾选JS分析结合burpsuite+HaE插件实现敏感信息提取。首先在burpsuite中设置proxy settings中代理端口，再点击反编译按钮会将反编译后的js代码代理到burpsuite中。

## 高级配置
在右侧面板可以配置以下参数：

- 输出目录 ：自定义反编译结果的保存位置
- 代码格式化 ：反编译时是否格式化代码，提高可读性
- JS分析 ：结合burpsuite+HaE插件实现敏感信息提取
- 分析端口 ：自定义分析端口用来分析多个小程序的情况，同时避免端口冲突
- 代理端口 ：代理服务的端口，与burpsuite代理端口一致即可
